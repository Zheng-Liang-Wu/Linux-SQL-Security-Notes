# 🔍 SQL 資料篩選 (Filtering with WHERE)

## 1. 基礎篩選：WHERE 子句
`WHERE` 關鍵字用於在查詢中建立篩選器，只回傳符合特定條件的紀錄。

* **語法：** `SELECT 欄位 FROM 表格 WHERE 條件;`
* **運算子：** 最常搭配等號 `=` 來進行精確比對。
* **分號位置：** 務必放在整句查詢（包含篩選條件後）的最末端。

---

## 2. 模式篩選：LIKE 運算子
當你不知道精確數值，只想搜尋「包含某種模式」的資料時，需使用 `LIKE` 搭配 **通配符 (Wildcards)**。

### 常用通配符 (Wildcards)
| 符號 | 說明 | 範例 | 匹配結果 |
| :--- | :--- | :--- | :--- |
| **`%`** | 代替**任意數量**的字元 | `'a%'` | apple123, art, a |
| **`_`** | **僅代替一個**字元 | `'a_'` | as, an, a7 |

### 更多比對範例
* `'a__'`：匹配所有以 'a' 開頭且總長度為 3 的字串 (如 ant, add)。
* `'%a'`：匹配所有以 'a' 結尾的字串 (如 pizza)。
* `'%a%'`：匹配所有**包含** 'a' 的字串 (如 Again, back)。
* `'_a_'`：匹配總長度為 3 且中間字元為 'a' 的字串 (如 Car, ban)。

---

## 💡 實戰範例 (Real-world Scenario)

假設你想在 `employees` 表格中，找出所有職稱包含 **"IT"** 的員工（例如 "IT 經理" 或 "IT 專員"）：

```sql
SELECT *
FROM employees
WHERE job_title LIKE 'IT%';
