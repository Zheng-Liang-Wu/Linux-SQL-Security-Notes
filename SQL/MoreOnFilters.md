# SQL 邏輯運算符 (Logical Operators)

邏輯運算符 `AND`、`OR` 和 `NOT` 用於在 `WHERE` 子句中結合多個條件，以精確篩選查詢結果。

---

## 1. AND 運算子
`AND` 用於根據**兩個或多個**條件進行篩選。只有當所有條件都為 **TRUE** 時，該記錄才會被包含在結果中。

* **特點**：所有條件必須同時滿足。
* **範例**：
    ```sql
    SELECT * FROM employees
    WHERE department = 'Sales' AND salary > 50000;
    ```

---

## 2. OR 運算子
`OR` 運算子連接兩個或多個條件，只要其中**任一條件**滿足（為 TRUE），結果就會被傳回。

* **特點**：滿足第一個、第二個或兩者皆滿足。
* **注意**：即使條件是基於同一列，也必須寫出**完整的條件陳述**。
* **範例**：
    ```sql
    -- 正確寫法
    SELECT * FROM customers
    WHERE country = 'Canada' OR country = 'USA';

    -- 錯誤寫法 (不可省略欄位名稱)
    -- WHERE country = 'Canada' OR 'USA'
    ```

---

## 3. NOT 運算子
`NOT` 運算子用來**否定**一個條件。它會傳回所有不符合指定條件的記錄。

* **特點**：只能處理單一條件（通常放在條件之前）。
* **替代語法**：尋找「不等於」某個值時，可以使用 `<>` 或 `!=`。

### 比較表：等效的篩選方式
| 語法類型 | 範例程式碼 |
| :--- | :--- |
| **使用 NOT** | `WHERE NOT country = 'USA'` |
| **使用 <>** | `WHERE country <> 'USA'` |
| **使用 !=** | `WHERE country != 'USA'` |

> **提示**：當查詢中同時使用 `AND` 與 `OR` 時，建議使用 **圓括號 `()`** 來明確指定運算的優先順序。
