# SQL 與資料庫術語集 (Glossary)

本筆記涵蓋了資料庫管理、資料類型及 SQL 查詢的核心術語。

---

## 🏗️ 基礎架構與系統
| 術語 | 定義 |
| :--- | :--- |
| **資料庫 (Database)** | 有組織的資料或資料集合。 |
| **關係型資料庫 (Relational Database)** | 一種結構化資料庫，其中包含相互關聯的表。 |
| **SQL (Structured Query Language)** | 結構化查詢語言，用於建立、互動及從資料庫請求資訊的程式語言。 |
| **查詢 (Query)** | 向資料庫表或多個表請求資料的操作。 |
| **文法 (Syntax)** | 決定電腦語言中正確結構化的規則。 |
| **日誌 (Log)** | 記錄組織系統內發生的事件。 |

---

## 🔑 關鍵字與關聯性
* **主鍵 (Primary Key)**：每行都有**唯一值**的列，用於識別唯一的記錄。
* **外鍵 (Foreign Key)**：一個表格中的欄位，是指向另一個表格中的**主鍵**，用於建立表與表之間的連結。

---

## 📊 資料類型 (Data Types)
* **字串資料 (String Data)**：由有序字元序列組成的資料（如文字、名稱）。
* **數值資料 (Numeric Data)**：由數字組成的資料，可用於數學計算。
* **日期和時間數據 (Date/Time Data)**：表示特定日期和/或時間的數據。

---

## 🔍 篩選與運算子
### 核心概念
* **篩選 (Filtering)**：選擇符合特定條件的數據。
* **運算子 (Operator)**：表示操作的符號或關鍵字（如 `+`, `=`, `AND`）。
* **通配符 (Wildcard)**：一種特殊字符，可以替換為任何其他字符（例如 SQL 中的 `%`）。

### 運算子分類
| 類型 | 定義 | 範例 |
| :--- | :--- | :--- |
| **包含運算符** | 包含比較值的運算子。 | `>=` (大於等於), `<=` (小於等於) |
| **互斥運算子** | 不包含比較值的運算子。 | `>` (大於), `<` (小於) |

---

> **學習筆記**：理解**主鍵**與**外鍵**的關係是掌握關係型資料庫（Relational Database）邏輯的最重要一步。
