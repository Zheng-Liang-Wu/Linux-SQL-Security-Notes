# 🐧 Linux 系統架構詳解

---

### 1. 系統層次與交互 (System Layers)

* **👤 使用者 (User)**
    * Linux 是一個**多用戶系統**，支援多個用戶同時使用與存取相同的資源。
* **📦 應用程式 (Applications)**
    * 執行特定任務的程式。
    * **軟體包管理器**：協助使用者安裝、管理與刪除軟體包的工具。
* **🐚 Shell (命令列解釋器)**
    * 系統的指令轉譯器，所有輸入至 Shell 的內容皆為**文字格式**。

---

### 2. 核心與檔案標準 (Kernel & Standards)

* **🧠 Kernel (核心)**
    * Linux 作業系統的核心組件，負責管理**進程 (Processes)** 與 **記憶體 (Memory)**。
* **📂 FHS (檔案系統層次結構標準)**
    * **Filesystem Hierarchy Standard**：定義 Linux 中組織資料與目錄結構的元件規範。

---

### 3. 硬體組件概覽 (Hardware Overview)

#### 🖥️ 內部硬體 (Internal Hardware)
> 電腦運作所需的關鍵核心組件。

| 組件名稱 | 縮寫 | 功能描述 |
| :--- | :--- | :--- |
| **中央處理器** | CPU | 電腦的主要處理器，執行通用計算任務。 |
| **隨機存取記憶體** | RAM | 用於**短期儲存**的硬體組件，供運算時快速存取。 |
| **硬碟** | Drive | 用於**長期儲存**的硬體組件，保存持久化資料。 |

#### 🖱️ 週邊設備 (Peripherals)
* **定義**：連接在電腦系統上並受其控制的硬體組件。
* **常見設備**：顯示器、印表機、鍵盤、滑鼠。
